# Student Performance Analysis Using Data Science

## Project Overview
This project analyzes student academic performance using data science and machine learning techniques.

## Features
- Data Cleaning
- Null Value Handling
- Duplicate Removal
- Outlier Detection
- Exploratory Data Analysis
- Correlation Matrix
- Linear Regression Model
- User Input Prediction System

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning
The project uses Linear Regression to predict student final scores based on:
- Study Hours
- Attendance
- Sleep Hours
- Family Support
- Internet Access
- Previous Scores
- Stress Level

## Output
The model predicts student final score and compares prediction with actual score.
